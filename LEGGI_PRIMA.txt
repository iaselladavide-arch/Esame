# Avviare il progetto (due terminali separati)

Terminal 1 (Backend):

  npm install
  npm start
  → Sarà su http://localhost:5000

Terminal 2 (Frontend):
  cd frontend
  npm install
  npm start
  → Si aprirà http://localhost:3000
