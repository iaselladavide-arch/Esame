## Come Eseguire Localmente

### Backend
```bash
cd backend
npm install
cp .env.example .env
node resetDatabase.js
npm start
```

### Frontend
```bash
cd frontend
npm install
npm start
```

## Testing API

Importa il file `Academy_Aziendale.postman_collection.json` per testare le API con Postman.
